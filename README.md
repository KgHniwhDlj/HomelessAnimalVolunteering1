# 🐾 Dайт волонтеров для помощи бездомным животным

Веб-приложение для автоматизации работы сотрудников и администраторов приюта. Позволяет регистрировать поступивших животных, отслеживать их состояние, статус и распределение по вольерам.
Модуль Администратора предоставляет возможность создавать новые вольеры и регистрировать новых сотрудников в систему, а так же редактировать данные как сотрудников, так и вольеров.

## 🛠 Технологии
* **HTML5** - структурирование страниц.
* **CSS3** - стилизация интерфейса.
* **JavaScript (ES6+)** - модульная архитектура, работа с DOM.
* **LocalStorage** - хранение данных (без внешнего сервера).

## 🚀 Функционал системы
* **Авторизация**: Разделение прав доступа на Администратора и Сотрудника.
* **Учет сотрудников**: Регистрация, валидация вводимых данных и автоматическое создание ID, редактирование и удаление.
* **Учет вольеров**: Создание, редактирование и удаление.
* **Учет животных**:
    * Регистрация карточки животного (кличка, вид, описание).
    * Динамический выбор вольеров из базы данных.
* **Умный поиск**: Поиск по всей базе данных (по кличке, ID, статусу, датам и даже имени волонтера).
* **Управление**: Смена вольеров, обновление медицинского состояния и статуса животного.

## 💾 Структура данных (LocalStorage)

Данные хранятся локально в браузере в формате JSON:
* `employee_[id]` - Профиль сотрудника (ФИО, email, пароль, роль).
* `animal_[id]` - Данные животного (связана по ID с создавшим сотрудником).
* `enclosure_[id]` - Список доступных вольеров.
* `current_user_id` - ID текущей активной сессии пользователя.
